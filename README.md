Shifty - Druid Rotation & Buff Helper

Version: 0.1.0

Features:
- Minimap button with drag support
- Settings panel split by forms
- Autobuff suggestions (Mark of the Wild, Thorns, Omen of Clarity)
- Suggestion frame (movable + lockable)

Commands:
/shifty toggle
/shifty show
/shifty hide
/shifty minimap
